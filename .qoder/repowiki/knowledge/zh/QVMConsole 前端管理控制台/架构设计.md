- 采用 Vue 3 + Vite 构建，使用 Pinia 进行全局状态管理（用户信息、VM 缓存），Vue Router 实现嵌套路由与权限守卫。
- 遵循分层架构：`src/api` 封装 Axios 请求并处理统一拦截（如高风险操作二次验证、Token 刷新）；`src/views` 负责页面逻辑；`src/components` 复用业务组件（如 VNC 控制台、监控图表）。
- 布局层 `layout/index.vue` 集成侧边栏导航、暗黑模式切换及近期任务面板，通过 `router-view` 动态渲染子视图。
- 依赖方向明确：视图层调用 Store 或 API 模块，API 模块依赖 `utils/request` 进行底层通信。
